# Security Policy

The following are all supported versions:

| Version | Supported          |
| ------- | ------------------ |
| 3.1.x   | :white_check_mark: |
| 3.0.x   | :x:                |

## Reporting a Vulnerability

If you encounter a vulnerability, please report it in the issues tab of this repository
